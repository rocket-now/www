よくある質問

ロケットナウに関するよくある質問をまとめました

サービスの営業時間は何時ですか？

ロケットナウの運営可能時間は午前8時〜翌日3時です。この時間内でお店の営業時間に合わせて設定することができます。カスタマーセンターは08:00-04:00まで対応しています。

受注タブレットはどのように操作しますか？

タブレットの操作は非常にシンプルです。ログイン後、新規注文が入ると通知が届き、注文内容の確認、調理時間の入力、準備完了の通知などの操作が簡単に行えます。また、メニューの売り切れ設定やレビュー確認などもセルフサービス画面から行うことができます。操作方法については、店舗を担当する営業スタッフが丁寧にご説明いたします。

複数の店舗を経営している場合の登録方法を知りたい

住所が同じであっても複数の店舗をロケットナウマネージャーに登録することができます。既存のログイン情報を使って、ロケットナウマネージャーにログイン後、「店舗追加」ボタンをクリックして新しい店舗を追加することができます。追加手続きは新規登録と同様の流れで進めることができます。

商品の価格やメニューは後から変更できますか？

はい、価格やメニューは後から変更可能です。売り切れや一時的に提供できない商品は、タブレットから簡単に非表示設定ができます。メニューの追加や価格変更などの本格的な変更については、メニュー修正依頼用のメールアドレス（rn-menu@rocketnow.co.jp）にご連絡いただければ対応いたします。

サービス開始までどのくらいかかりますか？

ロケットナウマネージャーの会員登録およびアカウント作成は代表者の同意後、5分以内に行えます。その後、タブレットの設置および最終的な本社からの案内電話までは5日以内に完了いたします。他社では2週間以上かかるのに比べ、ロケットナウはオープンまでのスピードが早く、店舗様から非常に高い満足度をいただいています。

店舗の精算はどのように行われるのですか？

月次精算が基本です。前月1か月間の売上が毎月10銀行営業日に支給されます。お問い合わせがあった場合には、週次精算も選択できます。週精算の場合、月曜日〜日曜日の売上が翌週火曜日に支給されます。

写真撮影に費用はかかりますか？

プロカメラマンによる撮影サービスを1回無料でご提供しています。メニューの魅力を最大限に引き出す高品質な写真をご用意いたします。撮影スケジュールは依頼日基準+5営業日で設定され、撮影日時と撮影メニュー数を事前にご相談させていただきます。

タブレットは無料で提供されますか？

はい、現在プロモーション期間中のため、タブレットを無料でレンタルしております。オペレーションにかかる費用は一切かかりません。

他のデリバリープラットフォームと違う点は何ですか？

ロケットナウは以下の点で他社と差別化しています：1) 送料無料のため新規顧客を創出する効果があります。2) 迅速なカスタマーサポート体制で、お問い合わせや問題に電話一本で対応できます。3) 天候が悪化した場合でも露出量が一定しており、デリバリーの売上を安定して確保できます。4) 業界最安水準の手数料と初期費用・固定費が一切無料という明確な料金体系です。
