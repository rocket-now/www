よくあるご質問

ロケットナウ出店に関するよくある質問をまとめました

ロケットナウの店舗会員登録はどこからできますか？

rocketnow Manager の WEBサイト もしくは、アプリ（Google Play / Apple Store）から可能です。

デリバリーサービスの営業時間は何時ですか？

営業可能時間は午前8時〜翌日3時です。この時間内でお店の営業時間に合わせて設定することができます。

売上はいつ支払われますか？

月次精算になります。毎月1日〜末日の月間売上が、翌月の10日（銀行営業日）に指定銀行口座へ支給されます。

誰が配達するのですか？

ロケットナウが委託した配達パートナーおよび配達業者が配達いたします。最短でお届けされるようマッチングされたドライバー（車、バイク、自転車）が注文品を受け取り、お客様にお届けいたします。

登録からサービス開始までどのくらいかかりますか？

およそ5日間で完了します。タブレットの設置訪問やメニュー情報の登録にお時間いただく場合がございます。

メニュー情報はどのように登録しますか？

他のフードデリバリーサービスを運用されている場合、メニュー情報の提出は特に不要です。そうでない場合は、出店申請完了後にメニュー情報の提出をご案内いたします。フォームもしくはエクセルにメニュー情報（メニュー名、価格、カテゴリー、説明文、写真）を記載いただくことになります。

メニュー情報を追加、変更する場合はどうしたらいいですか？

rocketnow Manager アプリからメニューの追加・変更が可能です。

お店が用意したタブレット端末でも運営できますか？

はい、roketnow Manager アプリを入れていただいた端末で運営いただけます。

タブレットの操作方法を知りたい

こちらのタブレットマニュアル動画をご確認ください。

手数料22%のプロモーションは自動で適用されますか?

はい、会員登録時に手数料35%と表示されている場合でも自動的に22%が適用されます。

通帳がない銀行口座でも登録できますか？

はい、登録可能です。通帳コピー画像の代わりに、口座情報が表示された画面のスクリーンショットをご提出ください。

複数の店舗を経営している場合の登録方法はどうなりますか？

rocketnow Manager で1店舗目をご登録いただいた後、店舗追加ボタンから2店舗目以降をご登録いただけます。出店申請は店舗ごとにフォームをご提出ください。
